Je leert informatie presenteren in een zelf vormgegeven interactieve omgeving. Deze week maak je een begin met het ontwerpen van een online portfolio. Verzamel inspiratie- en referentiemateriaal. Bekijk portfolio’s van alumni, influencers, media designers en toonaangevende bedrijven. Plaats de voorbeelden in een document en benoem kort wat je er goed of niet goed aan vindt.

De opdracht Uitleg over wat de opdracht inhield en hoe deze is uitgevoerd

De opdracht

Deze opdracht gaat over het maken van een online portfolio. In dit portfolio verwerken we alle eerder gemaakte opdrachten, met daarbij een reflectie. Als eerst ben ik gaan zoeken naar inspiratie voor een online portfolio. Deze ben ik online gaan googlen en heb ik genoteerd wat ik de pro's en cons vond van de website. Op basis van deze punten heb ik een goed idee hoe een fijn online portfolio er uit ziet. Ik merkte dat ik me er aan irriteerde als alle pagina's er anders uit zien en als er (veel) animaties op de pagina gebruikt worden.
Daarna heb ik een aantal schetsen gemaakt met potloot en papier hoe ik de layout van mijn website wil hebben. Tijdens het maken van deze layout heb ik nagedacht over duidelijkheid, locatie van informatie, verschillende schermmaten en niet over-de-top design.
Als laatst ben ik de website gaan maken. Omdat ik al ervaring heb met HTML en CSS ging dit vrij gemakkelijk. Ik heb op basis van mijn gemaakte design de website opgebouwd. Ook heb ik de website 'responsive' gemaakt voor telefoons.

Reflectie Mijn eigen reflectie op de opdracht

Tijdens het maken van dit portfolio heb ik eerst onderzoek gedaan naar al bestaande portfolio’s. Dit is een skill die je veel gebruikt bij media design en software development. Het bekijken van andersmans werk en hier de tips en tops uit halen zorgen er voor dat je eigen eindproduct een stuk beter kan worden. Dit heb ik ook gemerkt bij het maken van mijn website. Ik kwam er achter dat ik animaties hinderlijk vind en dat ik het fijner vind als een website een beetje simpel is.
